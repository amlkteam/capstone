1. after unzipping this folder, go to your command line terminal. (Windows user can use Windows PowerShell as the terminal)

2. in the terminal, change your working directory to the unzipped folder. i.e. Type "cd", space, then the unzipped/folder/absolute/path. 

3. Type "python dash_frontend_final.py"

4. a webpage frontend will run in the local server address provided to you, such as http://127.0.0.1:8050.  Please type in the address in your favourite browser to check out the visualization app.

Prerequisites/ Some python packages you need to install:

- pandas
- dash
- plotly
- scipy

Installation References:

- https://dash.plotly.com/installation
